Cómo conectar Notion (paso a paso)

1. Crear la integración en Notion:
Ve a notion.so/my-integrations → "New integration" → ponle nombre (ej: "Traductor") → copia el Internal Integration Token (empieza con secret_).

2. Conectar a tu base de datos:
Abre la base de datos en Notion → clic en los tres puntos ... → "Connect to" → selecciona tu integración. La base de datos necesita al menos estas propiedades: Name (título), Traducción (texto), Idioma (texto), Fecha (fecha).

3. Copiar el ID de la base de datos:
La URL de tu base de datos es así:
https://www.notion.so/TU_ID_AQUI?v=...
Copia solo la parte en negrita (32 caracteres).

4. Proxy CORS (necesario para archivos locales):
Notion bloquea solicitudes directas desde archivos HTML. Necesitas un intermediario. La opción más fácil es Make.com:

  1. Crea una cuenta gratis en make.com
  2. Crea un escenario con el trigger "Webhooks → Custom webhook"
  3. Agrega el módulo "Notion → Create a database item"
  4. Copia la URL del webhook y pégala abajo

Alternativa sin proxy: sube este archivo HTML a un servidor web (GitHub Pages, Netlify) — desde HTTPS la API de Notion funciona directamente.
